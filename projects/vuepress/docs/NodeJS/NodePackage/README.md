## nodemon
```bash
# 安装：全局安装
npm install -g nodemon

# 查看用法帮助
nodemon --help

# 查看版本
nodemon -v
nodemon --version

# 代替node启动js脚本
nodemon <script.js>
nodemon app.js

# 可设置端口号
nodemon app.js localhost 8888
```
