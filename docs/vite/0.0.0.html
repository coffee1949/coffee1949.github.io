千.0.0 vite介绍

vite

官方文档：https://vitejs.dev/

官方中文文档：https://cn.vitejs.dev/

特性
  • 1，极速的服务启动：完全类型化的API
  • 2，轻量快速的热重载：无论应用程序大小如何，都始终极快的模块热重载（HMR）
  • 3，丰富的功能：对TypeScript、JSX、CSS等支持开箱即用。
  • 4，优化的构建：可选 “多页应用” 或 “库” 模式的预配置Rollup构建
  • 5，通用的插件：在开发和构建之间共享Rollup-superset插件接口。
  • 6，完全类型化的API：灵活的API和完整的TypeScript类型。
